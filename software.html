Software


*** Please try to install the software before the course starts - this will make everything much quicker on the day. ***


You will need:

  • A laptop running Windows, OSX or Linux
  • Optionally, an image-editing program (Paint, Photoshop, GIMP etc - any will do!)
  • Stencyl and Processing installed. Instructions below.

Installing Stencyl

Stencyl is a deceptively simple but quite powerful game creation tool that allows you to create videogames for the desktop, web and mobile devices. It's very easy to use and install:

  1. Go to the Stencyl website and install the latest version. Stencyl is available for Windows, OSX and Linux. You should see a page similar to this, so choose your platform and install the software.

  2. That's it! Almost. There's one setting in Stencyl we'll need to change, which will adjust the artwork size when we make games.

  3. Open Stencyl, and go to File > Preferences.

  4. Go to the Workspace tab, and set the Scale to be "Standard (1x)". You'll only need to do this once and Stencyl will remember the setting as long as it's installed.



Installing Processing

Processing is an open-source creative coding environment, mostly designed for the arts and education. Happily, it's also excellent for learning to code simple games, and it's really simple to use - but again, is surprisingly powerful once you get your head around it. Installing this one is easy, just head to the Processing website and install it from their instructions.
