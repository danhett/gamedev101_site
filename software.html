Software

This page contains everything you'll need for GAMEDEV 101 on both levels, along with installation instructions.


*** Please try to install the software before the course starts - this will make everything much quicker on the day. ***


EVERYONE will need:

  • A laptop running Windows, OSX or Linux
  • An image-editing program (Paint, Photoshop, GIMP etc - any will do!)

Level 01 users will need:

  • Stencyl

Level 02 users will need:

  • Haxe and OpenFL (instructions below)
  • Sublime Text (ideally) or any plain-text editor. Please note that programs like Microsoft Word are *not* suitable for writing code!
  • A terminal or command prompt (instructions below)

Installing Stencyl

Stencyl is a powerful game creation tool that allows you to create videogames for the desktop, web and mobile devices. It's very easy to use and install:

  1. Go to the Stencyl website and install the latest version. Stencyl is available for Windows, OSX and Linux. You should see a page similar to this, so choose your platform and install the software.

  2. That's it! Almost. There's one setting in Stencyl we'll need to change, which will adjust the artwork size when we make games.

  3. Open Stencyl, and go to File > Preferences.

  4. Go to the Workspace tab, and set the Scale to be "Standard (1x)". You'll only need to do this once and Stencyl will remember the setting as long as it's installed.



Installing Haxe/OpenFL

Haxe is an open-source programming language, designed to let you create games and apps for almost every platform out there. OpenFL is a free Haxe library designed to make coding games easy and fun (OpenFL uses a tool called Lime to build projects). All three can be automatically installed by following these easy steps:

  1. Go to the OpenFL website and install the latest version of Haxe from here - there is an installer on this page for Windows, OSX and Linux.

  2. For the next part of setup (and later to compile our games) we're going to use the command line - this sounds really scary if you've not done it before, but it isn't, we promise! Using the command line is the best way to install and work with many things, and Haxe is one of them. So let's jump in!

    3. First, we'll need to open a terminal.

    • Windows users: go to the Start menu, then Accessories > Command Prompt.
    • Mac users: The easiest way is to click the serch box in the top right of your screen, and search for Terminal. Alternatively it's in Applications > Utilities > Terminal.
    • Linux users: this will vary depending on what you're using, but you can usually search for it, or hit Ctrl + Alt + T.

    The first thing we'll do is check that Haxe has been installed. Type the following into your terminal and hit Enter: 

    haxe -version

    You should see something like "3.1.3" - this means everything's working!

  3. Next, make sure you're connected to the internet and type:

    haxelib install openfl

    Your command prompt will show you the download progress, and will let you know when it's finished.

  4. Next, type in:

    haxelib run openfl setup

  5. And that's it! To check everything's worked, run:

    openfl

If you see a big OpenFL logo in your terminal, everything worked. If not, get in touch and we'll try to help (or if you're attending the course, let us know when you arrive and we'll help set you up.
